[yes-no-questions-hook] When asking the user a question with discrete choices, ask it as numbered yes/no questions (1./2./3.), one per line. Collapse any either/or into a single yes/no phrased around one branch ("no" = the other). Never embed "or" offering two courses of action; never list the branches as pick-one options. Use a pick-one list or AskUserQuestion only for 3+ genuinely mutually-exclusive branches that can't reduce to yes/no.
